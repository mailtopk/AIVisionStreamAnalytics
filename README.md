## AIVisionStreamAnalytics 

This is a modular video analytics pipeline developed in C++ on top of NVIDIA DeepStream. It enables real-time processing of video streams using multiple deep learning models, including YOLO-based object detection networks. The system is designed for scalable, high-throughput inference with efficient GPU acceleration, making it suitable for edge deployments and intelligent video applications. Development and testing have been performed on NVIDIA Jetson Orin Super, ensuring optimized performance on embedded GPU platforms.
